## Prerequisites

Before running the code, make sure you have python installed and the following packages with a compatible version with our code, as the followins:
- streamlit = 1.29.0
- pandas = 2.0.3
- matplotlib.pyplot = 3.3.4 
- seaborn = 0.11.1
- wordcloud = 1.9.3
- requests = 2.31.0
- numpy = 1.23.5
- shap = 0.44.0

## Execution
To run the code, go to the directory where Welcome.py is found and run the follwing command:

```bash
streamlit run Welcome.py
```
